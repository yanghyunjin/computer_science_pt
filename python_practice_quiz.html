Python Practice Quiz for Beginners

No starter code — write everything yourself! input · print · if · for

  1. Greeting Program
    Ask for the user’s name and print: "Hello, ___!"
    Hint: Use input() to ask; use print() to show the message.
  2. Age Checker
    Ask for an age. If the age is 13 or older, print “You’re a teenager!”, otherwise print “You’re still a kid!”.
    Hint: Use if / else; convert input to a number with int().
  3. Say Hello 5 Times
    Print “Hello!” five times in a row.
    Hint: Use a for loop with range(5).
  4. Odd or Even
    Print numbers 1 through 5. For each number, print “Even!” if it’s even, otherwise “Odd!”.
    Hint: % finds a remainder; == means “is equal to”; even check: num % 2 == 0.
  5. Repeat Your Name
    Ask for your name, then print it 3 times.
    Hint: Use input() to get the name and a for loop with range(3).
  6. Number Countdown
    Print the numbers from 5 down to 1.
    Hint: range(start, stop, step) with a negative step like -1.
  7. Favorite Color
    Ask for someone’s favorite color, then print a sentence like: “Wow, I love ___ too!”.
    Hint: Combine text and variables in print() (comma or +).
  8. Sum of 1 to 10
    Calculate and print the total of numbers from 1 through 10.
    Hint: Start with total = 0; use a for loop and keep adding.
  9. Multiplication Practice
    Ask for a number, then print its times table from 1× to 5×.
    Hint: Use int() to convert input; loop with range(1, 6).
  10. Word Length
    Ask the user to type any word, then print how many letters it has.
    Hint: Use len() to count characters.

Tip: Encourage students to save each solution in a separate file (e.g., q1.py, q2.py …) and run them from the terminal.
